# go-clicli

[![Build Status](https://github.com/cliclitv/go-clicli/workflows/build/badge.svg?branch=master)](https://github.com/cliclitv/go-clicli/actions)

[www.clicli.cc](https://www.clicli.cc)

### 依赖
* httprouter
* fasthttp
* go-sql-driver
* jwt
* jieba

### 特征
* 用户、番剧的增删改查
* 原生 sql 操作，包含外键关联、分页、模糊搜索
* 简单的 jwt 和数值鉴权
